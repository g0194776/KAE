############################################################
#
# KJFramework.Game.Components 项目是搭建游戏的快速解决方案，
# 通过将一个游戏拆分成若干部分，并且按照插榉木的方式进行组合。
#
# KJFramework.Game.Components 项目本身会提供若干组件， 这些组件
# 都是构建游戏的最小元素，使用者可以通过挑选，并改造组件来完成游戏的局部
# 或者全部的逻辑。
############################################################



*************************************************************************
*	2010年10月9日更新
*************************************************************************
1. 重写了TextBox, 简单的实现了录入，删除，以及光标移动和闪烁的基本功能。
2. 创建了Image类，用于显示静态的图片。
3. 创建了Label类，用于显示文字信息。
4. 创建了DisappearImage类，用于快速呈现游戏开头LOGO展示的效果。
5. 添加了一个内容处理器，用于支持中文字体的展示。


*************************************************************************
*	2010年10月12日更新
*************************************************************************
1. 构造了窗口的雏形类(Window)，作为一个窗口雏形，此窗口将会默认拥有一个背景和一个
   控件集合。
2. 重写了控件组
3. 重写了场景(仍然需要改善)
4. 重写了动作(Action)
5. 确定了“排版顺序”
   (1). 从选择动作开始
   (2). 每个动作中包含了若干个场景
   (3). 每个场景中包含了若干个组件，窗体，并内置一些事件操作，比如窗体移动等等。
   (4). 从场景中派生出一种游戏场景，这种游戏场景将包含如下内容：
		[1]. 地图加载器
		[2]. 角色控制
		[3]. 控制台
		[4]. 窗体控制器
		[5]. 物品控制器
		......

下面将会为每个组件进行实现。
